Lesson 7.6: Nested Arrays in JavaScript for AI

1. What Are Nested Arrays?

Nested arrays (arrays inside arrays) are often used to represent complex AI data like matrices, pixel values, or neural network layers.

let matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];

2. Accessing Elements

console.log(matrix[0][1]); // Output: 2 (first row, second column)

3. Iterating Over Nested Arrays

for (let row of matrix) {
  for (let value of row) {
    console.log(value);
  }
}

4. Modifying Nested Arrays

matrix[1][2] = 99;
console.log(matrix[1]); // Output: [4, 5, 99]

5. Use Case: Pixel Grid (AI Image Input)
