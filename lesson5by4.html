Lesson 5.4: Using do-while for Guaranteed Execution

What is a do-while loop?

The do-while loop runs the block of code once before checking the condition. It's ideal when you need to ensure the logic executes at least one time—like initializing input or processing default AI state.

Basic Syntax

let i = 0;
do {
  console.log("AI Iteration:", i);
  i++;
} while (i < 5);

AI Example: Simulated Input Until Valid

let input;
do {
  input = Math.floor(Math.random() * 10);
  console.log("AI Guess:", input);
} while (input !== 7);
console.log("AI found lucky number 7!");

Key Points

  • do-while guarantees at least one execution.
  • Useful in AI pre-processing, retry logic, and guaranteed initialization.
  • Be cautious of infinite loops—ensure the condition eventually becomes false.
