Lesson 8: Objects & Properties

Structure AI knowledge using objects and properties. Learn how to store and manipulate complex data.

Features of Objects & Properties

  • 1. **Creating Objects** - Learn how to define and initialize an object with properties.
  • 2. **Accessing Object Properties** - Use dot notation and bracket notation to retrieve values.
  • 3. **Modifying Properties** - Update properties using direct assignment.
  • 4. **Deleting Properties** - Remove properties from objects using `delete`.
  • 5. **Iterating Over Object Properties** - Use `for...in` loop to iterate through properties.
  • 6. **Nested Objects** - Create objects inside objects to represent more complex data.
  • 7. **Dynamic Property Names** - Set object keys dynamically using variables.
  • 8. **Computed Property Names** - Use computed property names to set dynamic keys.
  • 9. **Default Property Values** - Assign default values to object properties when initializing.
  • 10. **Using Object Methods** - Define methods inside objects and use `this` to refer to the object.
  • 11. **Accessing Methods in Objects** - Call methods defined within an object using dot notation.
  • 12. **Using `Object.create()`** - Create objects based on a prototype using `Object.create()`.
  • 13. **Prototypal Inheritance** - Leverage inheritance in objects to extend their functionality.
  • 14. **Using `Object.assign()` for Merging Objects** - Merge two or more objects into one using `Object.assign()`.
  • 15. **Using `Object.freeze()`** - Prevent modifications to an object by freezing it with `Object.freeze()`.
  • 16. **Using `Object.defineProperty()`** - Define object properties with more control over their behavior.
  • 17. **Getter and Setter Methods** - Control property access and modification using getter and setter functions.
  • 18. **Object Destructuring** - Use destructuring to extract values from objects into variables.
  • 19. **Using `Object.keys()`** - Get an array of an object's own property names.
  • 20. **Using `Object.values()`** - Get an array of an object's own property values.
  • 21. **Using `Object.entries()`** - Convert an object to an array of [key, value] pairs.
  • 22. **Nested Array of Objects** - Represent complex data with arrays of objects.
  • 23. **Handling Undefined Properties** - Safely access undefined properties to avoid runtime errors.
  • 24. **Shallow vs Deep Copy** - Copy objects and understand the difference between shallow and deep copies.
  • 25. **Using `in` Operator** - Check if a property exists in an object using the `in` operator.

Examples and Code

Example 1: Creating and Accessing an Object

const person = { name: 'John', age: 30, greet: function() { console.log('Hello ' + this.name); } }; console.log(person.name); // Output: John person.greet(); // Output: Hello John

Example 2: Modifying an Object

person.age = 31; console.log(person.age); // Output: 31

Example 3: Deleting a Property

delete person.age; console.log(person.age); // Output: undefined

Example 4: Nested Objects

const car = { make: 'Toyota', model: 'Corolla', owner: { name: 'Alice', age: 28 } }; console.log(car.owner.name); // Output: Alice

Summary

In **Lesson 8: Objects & Properties**, we've learned how to define and manipulate JavaScript objects. Objects are used to store data in a structured manner, and their properties can be dynamically accessed and modified. By using methods such as `Object.create()`, `Object.assign()`, and `Object.freeze()`, we can enhance the functionality of objects and prevent unintended side effects in our code. The examples provided showcase basic usage along with advanced techniques such as inheritance, getter/setter methods, and deep copying.
