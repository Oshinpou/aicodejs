Lesson 2.8 - Introduction to AI in JavaScript

Artificial Intelligence is not just for Python. JavaScript can also be used to build lightweight AI logic for the web. In this lesson, we'll explore how you can begin working with basic AI techniques in JavaScript.

Basic AI: Rule-Based Decisions

Simple AI systems use decision trees and logic conditions. Here's a basic example:

const userInput = "hello";
if (userInput.includes("hello")) {
 console.log("Hi there, how can I assist you today?");
}

Learning from Data: Pattern Matching

Using basic statistical methods or pattern matching, we can mimic a learning system:

const responses = {
 "hello": "Hi!",
 "how are you": "I'm a script, I feel no pain!",
 "bye": "Goodbye!"
};

function chatBot(input) {
 return responses[input.toLowerCase()] || "I don't understand that yet.";
}

console.log(chatBot("hello")); // Output: Hi!
