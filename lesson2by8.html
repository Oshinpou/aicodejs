Lesson 10: Events & Interaction

Learn how to make your AI code interactive with user interactions like clicks, key presses, and mouse movements.

Features of Events & Interaction

  • 1. Click Events

    Trigger actions when a user clicks on an element.

    
document.getElementById("myButton").addEventListener("click", function() {
  alert("Button clicked!");
});
  • 2. Mouseover Events

    Detect when the mouse hovers over an element.

    
document.getElementById("myDiv").addEventListener("mouseover", function() {
  this.style.backgroundColor = "yellow";
});
  • 3. Mouseout Events

    Detect when the mouse leaves an element.

    
document.getElementById("myDiv").addEventListener("mouseout", function() {
  this.style.backgroundColor = "";
});
  • 4. Keydown Events

    Capture keypress events when any key is pressed.

    
document.addEventListener("keydown", function(event) {
  console.log("Key pressed: " + event.key);
});
  • 5. Keyup Events

    Capture key release events when a key is released.

    
document.addEventListener("keyup", function(event) {
  console.log("Key released: " + event.key);
});
  • 6. Mousemove Events

    Track mouse movement within the page.

    
document.addEventListener("mousemove", function(event) {
  console.log("Mouse at: " + event.clientX + ", " + event.clientY);
});
  • 7. Mouseenter Events

    Fire when the mouse enters an element.

    
document.getElementById("myElement").addEventListener("mouseenter", function() {
  console.log("Mouse entered the element.");
});
  • 8. Mouseleave Events

    Fire when the mouse leaves an element.

    
document.getElementById("myElement").addEventListener("mouseleave", function() {
  console.log("Mouse left the element.");
});
  • 9. Touchstart Events

    Detect the start of a touch interaction on mobile devices.

    
document.getElementById("myTouchArea").addEventListener("touchstart", function(event) {
  console.log("Touch started");
});
  • 10. Touchend Events

    Detect the end of a touch interaction on mobile devices.

    
document.getElementById("myTouchArea").addEventListener("touchend", function(event) {
  console.log("Touch ended");
});
