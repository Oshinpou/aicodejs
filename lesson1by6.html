Lesson 1.6 – Functions

In this lesson, you'll learn how to define and call functions like a pro in JavaScript. Functions are essential for writing reusable and clean code.

What is a Function?

A function is a block of code that performs a specific task and can be reused.

function greet(name) {
  return "Hello, " + name + "!";
}

console.log(greet("AI Student"));

Function Declaration vs Expression

// Declaration
function add(x, y) {
  return x + y;
}

// Expression
const multiply = function(x, y) {
  return x * y;
};

Arrow Functions

Introduced in ES6 for a shorter syntax.

const divide = (x, y) => x / y;

console.log(divide(10, 2));
