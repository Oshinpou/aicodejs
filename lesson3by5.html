Lesson 3.5: Introduction to ES6

Explore modern JavaScript features and syntax.

Overview

ECMAScript 6 (ES6) introduced many new features that significantly improve JavaScript development. Let's go over some core features.

Key ES6 Features

  • Let & Const: Block-scoped variable declarations
  • Arrow Functions: Compact syntax for function expressions
  • Template Literals: Multi-line strings and embedded expressions
  • Destructuring: Extract values from arrays or objects
  • Default Parameters: Default values in function parameters
  • Spread & Rest: Spread operator and rest parameters
  • Classes: Cleaner way to create objects and inheritance

Task

Create a function using arrow syntax that accepts two parameters and returns their sum. Use const for declaration.
