Lesson 15.3 — Local Scope in AI

Keep AI logic clean with isolated variables in JavaScript.

What is Local Scope?

Variables declared inside a function or block using let or const are said to be in local scope. They can't be accessed outside their defining block.

Example: Local Scope in AI Function

function aiTask() {
  let task = "training model";  // Local variable
  console.log("Inside AI task:", task);
}

aiTask();
console.log("Outside AI task:", task); // Error: task is not defined

Why Local Scope Matters in AI

Local scope helps prevent variable collisions and bugs in large AI applications. Isolating data keeps your logic modular and secure, especially when handling sensitive data or multiple AI processes.
