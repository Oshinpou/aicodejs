Lesson 1.3: Operators & Expressions

Operators perform operations on variables and values. Expressions combine variables and operators to compute values.

Types of Operators

  • Arithmetic: +, -, *, /, %
  • Assignment: =, +=, -=, *=, /=
  • Comparison: ==, ===, !=, !==, >, <, >=, <=
  • Logical: &&, ||, !

Example Code

// Arithmetic operations
let a = 10;
let b = 5;
console.log(a + b);  // 15
console.log(a * b);  // 50// Comparison console.log(a > b);  // true console.log(a === 10); // true
