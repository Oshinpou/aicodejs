Lesson 3: Operators & Expressions

Master JavaScript logic to control AI behavior with operators and expressions.

Code Editor

Examples & Explanations

Example 1: Addition Operator (+) 
let sum = 5 + 3; console.log(sum);

Explanation: Adds two numbers and returns the result.

Example 2: Subtraction Operator (-) 
let difference = 10 - 7; console.log(difference);

Subtracts the right operand from the left one.

Example 3: Multiplication (*) 
let product = 4 * 2; console.log(product);
Example 4: Division (/) 
let quotient = 20 / 5; console.log(quotient);
Example 5: Modulus (%) 
let remainder = 10 % 3; console.log(remainder);
Example 6: Exponentiation (**) 
let power = 2 ** 3; console.log(power);
Example 7: Assignment (=) 
let a = 10;
Example 8: Comparison (==, ===, !=, !==) 
console.log(5 == "5"); // true
console.log(5 === "5"); // false
Example 9: Logical Operators (&&, ||, !) 
console.log(true && false); // false
console.log(true || false); // true
Example 10: Ternary Operator 
let age = 18; let canVote = age >= 18 ? "Yes" : "No";

Important Notes

"===" checks value and type strictly.
"==" checks only value, not type.
Modulus (%) gives the remainder.
Logical NOT (!) flips true to false.
Ternary is a shortcut for if-else conditions.
