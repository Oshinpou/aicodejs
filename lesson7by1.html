Lesson 7.1: Introduction to Arrays

1. What is an Array?

Arrays are lists used to store multiple values in a single variable. They are especially useful in AI for storing datasets, prediction results, or sequences of operations.

let predictions = [0.1, 0.45, 0.89, 0.99];

2. Accessing Array Elements

let aiScores = [82, 90, 88];
console.log(aiScores[0]); // Output: 82

3. Modifying Arrays

let data = [1, 2, 3];
data[1] = 5; // data becomes [1, 5, 3]

4. Array Methods for AI

let inputs = [5, 7, 9];
inputs.push(11);       // Add new data
inputs.pop();          // Remove last
let sum = inputs.reduce((a, b) => a + b);
