Lesson 8.2: Object Methods and Functions

1. What Are Object Methods?

Methods are functions stored as properties inside an object. They help define object behavior, which is useful in AI systems for actions like training, prediction, or resetting models.

2. Defining a Method Inside an Object

let aiModel = {
  name: "SentimentAnalyzer",
  train: function() {
    console.log("Model is now training...");
  }
};

aiModel.train();

3. Using the `this` Keyword

let aiBot = {
  name: "NeuroBot",
  status: "idle",
  activate: function() {
    this.status = "active";
    console.log(this.name + " is now " + this.status);
  }
};

aiBot.activate();
