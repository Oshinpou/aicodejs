Lesson 10.4: Mouse Events & AI Movement

Mouse events let us track user movement, which is useful for creating interactive AI models or simulations that respond to user behavior.

Example: AI Follows the Mouse

JavaScript Code


const playground = document.getElementById("playground");
const aiBall = document.getElementById("aiBall");

playground.addEventListener("mousemove", function(e) {
  const rect = playground.getBoundingClientRect();
  const x = e.clientX - rect.left - 15;
  const y = e.clientY - rect.top - 15;
  aiBall.style.left = x + "px";
  aiBall.style.top = y + "px";
});
