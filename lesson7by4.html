1. Why These Methods Matter in AI

These built-in array methods are perfect for processing data efficiently in AI workflows such as transforming input data, filtering predictions, and analyzing datasets.

2. `forEach()` - Process Each Element

let activations = [0.3, 0.7, 0.9];
activations.forEach((val, index) => {
  console.log("Neuron", index, "fired with", val);
});

3. `map()` - Transform Values

let normalized = activations.map(val => val * 100);
// Output: [30, 70, 90]

4. `filter()` - Select Relevant Values

let confident = activations.filter(val => val > 0.5);
// Output: [0.7, 0.9]
