Lesson 8.5: Working with Nested Objects

1. What Are Nested Objects?

Nested objects are objects that contain other objects as properties. This structure is useful for modeling complex AI data like neural layers, configurations, or model metadata.

2. Example: AI Model with Nested Properties

const aiModel = {
  name: "TextGen",
  version: "1.0",
  metadata: {
    created: "2025-04-01",
    author: "AI Labs",
    training: {
      dataset: "CustomCorpus",
      epochs: 10
    }
  }
};

console.log(aiModel.metadata.training.dataset);

3. Updating Nested Object Values

aiModel.metadata.training.epochs = 20;
console.log(aiModel.metadata.training);
