Lesson 10.8: Event Delegation for Smart Handling

Use event delegation to handle AI-generated elements efficiently.

Waiting for AI interaction...

JavaScript Code


const container = document.getElementById("ai-container");
const output = document.getElementById("output");

// Event delegation: handle all buttons from one parent
container.addEventListener("click", function(event) {
  if (event.target.tagName === "BUTTON") {
    const action = event.target.getAttribute("data-ai");
    output.textContent = \`AI triggered: Action \${action}\`;
  }
});
