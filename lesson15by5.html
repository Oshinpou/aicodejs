Lesson 15.5 — Nested Scope (Scope Chains)

Follow the chain of AI variable access through functions.

What is Nested Scope?

In JavaScript, functions can be nested inside other functions. Inner functions have access to the variables of their outer functions. This structure is called the scope chain.

Example: AI Variable Scope Chain

function aiCore() {
  const core = "Main AI";

  function aiModule() {
    const module = "Vision";

    function aiTask() {
      const task = "Face Recognition";
      console.log(core);    // Accessible
      console.log(module);  // Accessible
      console.log(task);    // Accessible
    }

    aiTask();
  }

  aiModule();
}

Note: Each inner function can access variables from its parent scopes, forming a "chain" of access.

Why It Matters in AI

Understanding nested scope helps in building modular AI components where inner layers (like decision engines or preprocessors) rely on shared logic from outer scopes.
