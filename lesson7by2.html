Lesson 7.2: Multi-Dimensional Arrays

1. What is a Multi-Dimensional Array?

Multi-dimensional arrays are arrays of arrays. In AI, they are commonly used for representing matrices, image grids, and input-output layers in neural networks.

let matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];

2. Accessing Values

Access elements using two indexes: array[row][column]

console.log(matrix[0][2]); // Output: 3

3. Looping Through Multi-Dimensional Arrays

for (let i = 0; i < matrix.length; i++) {
  for (let j = 0; j < matrix[i].length; j++) {
    console.log(matrix[i][j]);
  }
}

4. AI Use Cases

  • Representing grayscale images (2D pixel arrays)
  • Storing AI training examples in matrix format
  • Building connection weights in neural networks
