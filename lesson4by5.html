Lesson 4.5: Combining if, else, and switch for Complex Decisions

Concept

Sometimes, using just if-else or switch alone isn't enough. In AI, we often deal with multi-layered decision-making where both can be combined.

Example Scenario

Determine an AI response based on user role and command:

let role = "admin"; let command = "shutdown";if (role === "admin") { switch (command) { case "start": console.log("AI: Starting systems..."); break; case "shutdown": console.log("AI: Shutting down now."); break; default: console.log("AI: Unknown admin command."); } } else { if (command === "status") { console.log("AI: System is running."); } else { console.log("AI: Permission denied."); } }

Use Cases in AI
