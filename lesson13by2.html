Lesson 13.2 — Mastering setTimeout()

Control time-based actions in AI-driven interfaces

What is setTimeout()?

setTimeout() is a built-in JavaScript function used to delay the execution of a function or code snippet.

Syntax:

setTimeout(function, delayInMilliseconds)

Advanced Use Case: Canceling Timeouts

You can cancel a timeout with clearTimeout().


const id = setTimeout(() => {
  console.log("This might not run!");
}, 5000);
clearTimeout(id);

This is useful if the user cancels a request before the AI completes it.
