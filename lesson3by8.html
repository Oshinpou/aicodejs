Lesson 3.8: Combining Operators in AI Applications

Introduction

Combining multiple operators enables AI programs to make powerful logical decisions, optimize conditions, and process data effectively. This lesson teaches you how to combine arithmetic, logical, and comparison operators.

Example 1: Arithmetic and Comparison

let age = 25;
let minAge = 18;
let maxAge = 30;
let isEligible = (age >= minAge) && (age <= maxAge);
console.log("Eligible:", isEligible);
Output: Eligible: true

Example 2: Logical and Ternary

let user = "admin";
let loggedIn = true;
let access = (user === "admin" && loggedIn) ? "Granted" : "Denied";
console.log("Access:", access);
Output: Access: Granted

Task

Create a program that checks if a user has a score between 70 and 100 and is not banned. Return "Pass" or "Fail" accordingly.

Quiz

What will this return? 
let a = 5, b = 10, c = 15;
console.log((a + b > c) || (b > c));
Answer: false

Advanced Practice

Use combined operators to create a smart AI alert system: 
let temperature = 35;
let humidity = 70;
let alert = (temperature > 30 && humidity > 60) ? "Trigger Cooling System" : "Normal Conditions";
console.log(alert);
Output: Trigger Cooling System

Summary

  • Combining operators increases decision-making power.
  • Use parentheses to control precedence.
  • Mix arithmetic, logical, comparison, and ternary operators safely.
