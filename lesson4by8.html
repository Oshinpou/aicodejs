Lesson 4.8: Introduction to Local Storage

Learn how to store data in the browser using Local Storage.

What You'll Learn:

  • What Local Storage is
  • How to save and retrieve data in the browser
  • Use cases for Local Storage

What is Local Storage?

Local Storage is a web storage object that allows you to store key-value pairs in a web browser. It saves data even when the browser is closed and reopened.

Storing Data

You can store data using localStorage.setItem(key, value):


localStorage.setItem('username', 'john_doe');

Retrieving Data

Retrieve data with localStorage.getItem(key):


let user = localStorage.getItem('username');
console.log(user); // Output: john_doe

Removing Data

Use localStorage.removeItem(key) to delete specific data:


localStorage.removeItem('username');

Clearing All Data

You can clear all Local Storage data with:


localStorage.clear();

Use Cases

  • Saving user preferences (dark/light mode)
  • Remembering form inputs
  • Storing login state (for simple apps)
