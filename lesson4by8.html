Lesson 4.8: Debugging and Troubleshooting Conditional Logic

Conditional statements can be tricky. In this lesson, you will learn how to debug your if, else, and switch logic to ensure your AI logic behaves as expected.

Common Issues in Conditional Logic

  • Using = instead of == or ===
  • Forgetting break in switch cases
  • Logical conditions that always return true or false
  • Misplaced curly braces {}
TIP: Use console.log() generously to check the values of variables inside conditional blocks.

Example: Debug This Code

let score = "80"; if (score = 80) { console.log("You passed!"); } else { console.log("Try again."); }

What's wrong? The condition uses = instead of == or ===. It should be:

let score = "80"; if (score == 80) { console.log("You passed!"); } else { console.log("Try again."); }
