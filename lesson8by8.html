Lesson 8.8: Working with Object Methods for AI Models

1. What Are Object Methods?

Object methods are functions defined inside objects. In AI, we use them to perform tasks like training, predicting, or evaluating AI models stored as objects.

const aiModel = {
  name: "TextClassifier",
  train() {
    console.log("Training model:", this.name);
  },
  predict(input) {
    console.log("Predicting with input:", input);
    return input.length > 10 ? "Positive" : "Negative";
  }
};

aiModel.train();
console.log(aiModel.predict("This is a test"));

2. The Role of this in AI Object Methods

The keyword this refers to the current object. It helps access the model's properties from within its own methods.
