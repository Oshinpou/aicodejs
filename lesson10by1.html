Lesson 10.1: What Are Events in JavaScript?

Events are actions that happen in the browser – like clicking a button, pressing a key, or moving a mouse. In JavaScript, you can listen for these events and run code in response – critical for AI interfaces that rely on user input or dynamic interaction.

Why Events Matter in AI

  • Trigger predictions when a user submits data.
  • Start or pause training loops with a button.
  • Dynamically update the UI based on sensor input or AI decisions.

Example: Button Click Event

AI Activated

// JavaScript Event Listener
document.getElementById("aiTrigger").addEventListener("click", function() {
  document.getElementById("output").style.display = "block";
  document.getElementById("output").textContent = "AI: Decision triggered!";
});
