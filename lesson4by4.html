Lesson 4.4: Introduction to the switch Statement

What is a switch Statement?

The switch statement provides a cleaner way to test a single variable against multiple values. Itâ€™s often easier to read than a long chain of else if statements, especially in AI decision trees.

Syntax

switch (expression) {
  case value1:
    // code block
    break;
  case value2:
    // code block
    break;
  default:
    // default code block
}

Example: AI Command Handler

let command = "status";
switch (command) {
  case "start":
    console.log("AI: Starting system...");
    break;
  case "stop":
    console.log("AI: Stopping system...");
    break;
  case "status":
    console.log("AI: System is operational.");
    break;
  default:
    console.log("AI: Unknown command.");
}
