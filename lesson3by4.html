Lesson 3.4: Logical Operators for AI Control

Introduction

Logical operators are essential in JavaScript to control the flow of logic, especially when programming AI behavior. These include && (AND), || (OR), and ! (NOT).

Logical Operators Overview

  • && - Returns true if both operands are true.
  • || - Returns true if one or both operands are true.
  • ! - Reverses the boolean value of its operand.

Example 1: Using AND (&&)

let isSmart = true;
let isFast = true;
if (isSmart && isFast) {
  console.log("AI is smart and fast!");
}
Output: AI is smart and fast!

Example 2: Using OR (||)

let hasBattery = false;
let isPluggedIn = true;
if (hasBattery || isPluggedIn) {
  console.log("AI can operate!");
}
Output: AI can operate!

Example 3: Using NOT (!)

let isOffline = false;
if (!isOffline) {
  console.log("AI is online!");
}
Output: AI is online!

Applications in AI

  • Conditional decision making
  • Controlling behavior trees
  • Validating inputs
  • Combining multiple checks
  • Triggering fallback logic
