Lesson 3.4: DOM Manipulation

Interact with web pages using JavaScript

What is the DOM?

The Document Object Model (DOM) represents the page content as a tree of objects. JavaScript can use this to read and modify content, structure, and styling.

Common Methods

  • getElementById() - selects an element by ID
  • querySelector() - selects the first matching element
  • createElement() - creates a new HTML element
  • appendChild() - adds an element to the DOM
  • innerHTML - gets or sets HTML content

Example

document.getElementById("demo").innerHTML = "Hello, DOM!";

const newPara = document.createElement("p"); newPara.textContent = "This is a new paragraph."; document.body.appendChild(newPara);
