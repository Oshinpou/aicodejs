Lesson 11.1: Introduction to Forms in JavaScript

This lesson introduces how JavaScript interacts with forms to collect user or AI input.

Form result will appear here.

JavaScript Code


const form = document.getElementById("aiForm");
const result = document.getElementById("result");

form.addEventListener("submit", function(e) {
  e.preventDefault();
  const name = form.name.value;
  const level = form.level.value;
  result.textContent = \`AI Model: \${name}, Intelligence Level: \${level}\`;
});
