Lesson 7.3: Iterating with for Loop

1. Why Use a for Loop in AI?

The for loop is essential in AI programming for processing data structures such as arrays, model weights, features, and training records.

2. Basic Syntax

for (let i = 0; i < data.length; i++) {
  console.log(data[i]);
}

3. Example: Summing AI Scores

let scores = [0.86, 0.74, 0.93, 0.67];
let total = 0;
for (let i = 0; i < scores.length; i++) {
  total += scores[i];
}
console.log("Total:", total);

4. AI Use Cases

  • Iterating over feature vectors
  • Summing model predictions
  • Processing tokenized input for NLP
  • Analyzing dataset rows
