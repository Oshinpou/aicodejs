Lesson 1: Getting Started

Learn the basics of JavaScript in AI. Set up your environment and start coding your first program!

Lesson Overview

In this lesson, you will set up your JavaScript development environment and write your first JavaScript program. By the end of this lesson, you'll understand how JavaScript interacts with AI systems, and you'll have written a simple AI program!

Getting Started with Your First JavaScript Program

To begin with, you'll need to open a browser console or use an online code editor like CodePen or JSFiddle. JavaScript code is executed in the browser, which makes it easy to test and see results immediately.

1. Example Task: Print "Hello World" to the Console

In JavaScript, the simplest way to interact with the user is by printing text to the console. Let's write the first piece of code:

console.log("Hello World");

Output: When you run this code, you should see:

Hello World

2. Example Task: Create a Variable

In JavaScript, variables are used to store data. You can create variables using the let or const keyword. Try the following example:

let greeting = "Hello, AI!";

Now print the variable:

console.log(greeting);

Output: The console will display:

Hello, AI!

3. Example Task: Basic Math with Variables

In JavaScript, you can also perform mathematical operations with variables:

let num1 = 5;
let num2 = 10;
let sum = num1 + num2;
console.log(sum);

Output: This will print:

15

4. Example Task: Using Functions

In JavaScript, functions allow you to organize your code into reusable blocks. Here's how to define and call a function:

function sayHello(name) {
  console.log("Hello, " + name + "!");
}
sayHello("AI");

Output: The console will display:

Hello, AI!

6. Example Task: Strings & Concatenation

Learn how to join strings together using concatenation:

let firstName = "John";
let lastName = "Doe";
let fullName = firstName + " " + lastName;
console.log(fullName);

Output: This will display:

John Doe

7. Example Task: Boolean Logic

Let's explore Boolean logic. A Boolean is either true or false. Here's an example:

let isAI = true;
if (isAI) {
  console.log("AI is cool!");
} else {
  console.log("AI is not cool.");
}

Output: If the condition is true, the console will display:

AI is cool!

8. Example Task: Working with Arrays

Arrays store multiple values. You can create and use them like this:

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // Apple

Output: This will display:

Apple

9. Example Task: Loops in JavaScript

Loops help you repeat tasks. Let's use a for loop to print all items in an array:

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

Output: This will display:

Apple
Banana
Cherry

In this lesson, you've learned the following:

  • How to write and run JavaScript code in a browser.
  • Basic concepts like variables, functions, strings, arrays, and loops.
  • How JavaScript can be used in AI to manipulate data, make decisions, and interact with the user.
