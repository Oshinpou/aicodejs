Learn Coding with Easy Examples

1. Hello World (HTML)

<h1>Hello World!</h1>

This example displays a simple heading in HTML.

2. Print Text (JavaScript)

console.log("Hello, World!");

This logs a message to the browser console.

3. Change Text (HTML + JavaScript)


<p id="demo">Original Text</p>
<button onclick="document.getElementById('demo').innerHTML = 'Text Changed!'">Change Text</button>

This changes the paragraph text when you click the button.

4. Simple Function (JavaScript)


function greet(name) {
    return "Hello, " + name + "!";
}
console.log(greet("Alex"));

This function greets the person by name.

5. Basic If Statement (JavaScript)


let age = 18;
if (age >= 18) {
    console.log("You are an adult.");
}

This checks if someone is an adult based on age.

6. Create a List (HTML)


<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Cherry</li>
</ul>

This creates a simple unordered list.

7. Simple Loop (JavaScript)


for (let i = 1; i <= 5; i++) {
    console.log(i);
}

This prints numbers 1 to 5 using a loop.

8. Create a Simple Table (HTML)


<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
</table>

This creates a simple table with name and age.

9. Simple Alert Box (JavaScript)

alert("Welcome to the site!");

This shows a popup alert message.

10. Basic CSS Styling


<style>
p {
    color: blue;
    font-size: 20px;
}
</style>
<p>Styled Text</p>

This styles the paragraph text to be blue and larger.
