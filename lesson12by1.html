Lesson 12.1: What is JSON?

JSON (JavaScript Object Notation) is a lightweight format for exchanging data between applications. It's especially popular in AI for data representation, predictions, configurations, and training logs.

Why AI Uses JSON

  • Standardized and easy to read/write
  • Fast and lightweight
  • Works across languages and systems

Example JSON

{
  "name": "Imacx AI Assistant",
  "skills": ["language understanding", "data processing", "prediction"],
  "active": true,
  "version": 1.5
}

JavaScript and JSON

Parsing JSON:
let jsonData = '{"name": "AI", "active": true}';
let obj = JSON.parse(jsonData);
console.log(obj.name); // Output: "AI"
Stringifying JS Object:
let jsObj = { model: "GPT", version: 4.0 };

let jsonString = JSON.stringify(jsObj); console.log(jsonString); // Output: '{"model":"GPT","version":4}'

AI Use Case Example

A machine learning model may return a JSON object like:

{
  "prediction": "cat",
  "confidence": 0.983
}
