Lesson 4.2: Using if, else for Simple Decisions

What is if-else?

In AI logic, decisions are made based on conditions. The if statement tests a condition. If true, it runs code; otherwise, the else block runs.

Syntax

if (condition) {
  // code if condition is true
} else {
  // code if condition is false
}

Example: AI Reaction Based on Score

let score = 75;
if (score >= 60) {
  console.log("AI: You passed!");
} else {
  console.log("AI: You need more practice.");
}

This script checks if a student passed a test based on score.
