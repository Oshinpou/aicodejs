Lesson 4.3: Exploring the else if Statement

What is else if?

When you want to test more than two conditions, else if allows your code to branch into multiple possibilities. It's very useful in AI to categorize or score things based on multiple thresholds or states.

Syntax

if (condition1) {
  // code if condition1 is true
} else if (condition2) {
  // code if condition2 is true
} else {
  // code if none of the above
}

Example: AI Mood Detection

let mood = "neutral";
if (mood === "happy") {
  console.log("AI: You seem cheerful today!");
} else if (mood === "sad") {
  console.log("AI: I'm here if you need a friend.");
} else {
  console.log("AI: Let's figure out how you're feeling.");
}
