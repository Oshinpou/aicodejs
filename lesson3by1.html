What is the DOM?

The Document Object Model (DOM) represents the structure of an HTML document as a tree of objects. You can use JavaScript to select and modify these objects in real-time.

Selectors

You can select elements using methods like getElementById, querySelector, and getElementsByClassName.

// Select an element

const title = document.getElementById('main-title'); title.textContent = "Welcome to AI!";

Modifying Content

You can change content and style dynamically using JavaScript:

document.querySelector(".subtitle").style.color = "blue";

document.querySelector(".subtitle").textContent = "AI is Learning You.";

Creating Elements

const newDiv = document.createElement("div");

newDiv.textContent = "I'm a new AI-generated element!"; document.body.appendChild(newDiv);
