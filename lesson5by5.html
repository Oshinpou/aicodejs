Lesson 5.5: Looping Through AI Data Sets

Why Loop Through Data?

AI applications often involve processing arrays and objectsâ€”like predictions, datasets, or model parameters. Looping allows us to work with each item efficiently.

Using for with Arrays

const predictions = [0.92, 0.76, 0.81, 0.64];
for (let i = 0; i < predictions.length; i++) {
  console.log("Prediction", i + 1, "=", predictions[i]);
}

Using for...of with Arrays

const inputs = ["image1.jpg", "image2.jpg", "image3.jpg"];
for (const img of inputs) {
  console.log("Analyzing:", img);
}

Using for...in with Objects

const modelStats = {
  accuracy: "92%",
  loss: 0.08,
  epochs: 10
};

for (const key in modelStats) {
  console.log(key + " =>", modelStats[key]);
}

Key Takeaways

  • for and for...of are great for arrays.
  • for...in works well with objects (key-value pairs).
