Level 2: Variables & Intro to JavaScript for AI

Learn the foundation of JavaScript to build smart AI logic

What are Variables?

Variables are containers for storing data values. They help AI remember, calculate, and make decisions.

let aiName = "Imacx AI";
const maxScore = 100;
var isTrained = true;

Types of Variables

  • var — Function-scoped, older style.
  • let — Block-scoped, modern and flexible.
  • const — Block-scoped, constant values only.
