Lesson 8.1: Introduction to JavaScript Objects

1. What Are JavaScript Objects?

Objects are collections of key-value pairs. In AI, they are used to structure data like model parameters, user inputs, or configuration settings.

2. Creating an Object

// Creating a basic AI model object
let aiModel = {
  name: "EmotionDetector",
  accuracy: 0.93,
  isTrained: true
};

3. Accessing Object Properties

console.log(aiModel.name); // "EmotionDetector"
console.log(aiModel['accuracy']); // 0.93

4. Modifying and Adding Properties

aiModel.accuracy = 0.95;
aiModel.version = "v2.0";
