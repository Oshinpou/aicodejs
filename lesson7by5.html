Lesson 7.5: Array Sorting and Searching

1. Why Sorting and Searching Matter in AI

Sorting helps organize AI model outputs, ranking probabilities, and structuring data. Searching enables efficient identification of important values such as predictions, scores, and threshold-based results.

2. Sorting Arrays

// Sorting confidence scores in ascending order
let scores = [0.92, 0.4, 0.85, 0.7];
scores.sort((a, b) => a - b);
console.log(scores); // [0.4, 0.7, 0.85, 0.92]

3. Searching in Arrays

// Finding index of specific confidence score
let targetIndex = scores.indexOf(0.85); 
console.log(targetIndex); // 2

// Check if score exists
console.log(scores.includes(0.4)); // true

4. Finding Values with Conditions

// Find the first value above threshold
let high = scores.find(val => val > 0.8);
console.log(high); // 0.85

// Get index of first high value
let highIndex = scores.findIndex(val => val > 0.8);
console.log(highIndex); // 2
