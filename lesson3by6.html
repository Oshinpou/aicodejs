Lesson 3.6 - DOM Manipulation

Interact with the webpage using JavaScript

What is the DOM?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It allows JavaScript to access and manipulate the content, structure, and style of a website dynamically.

Example

// Selecting an element

const heading = document.getElementById("myHeading");

// Changing its text heading.textContent = "Hello from JavaScript!";

// Adding a class heading.classList.add("highlight");
