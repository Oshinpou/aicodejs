Lesson 5.1: Introduction to Loops

Loops allow you to repeat a block of code multiple times, making your code more efficient and powerful. JavaScript supports different types of loops: for, while, and do...while.

Why Loops Matter in AI

In AI programming, loops are used to train models, process datasets, or repeat actions until a condition is met.

Example: Simple For Loop

for (let i = 0; i < 5; i++) {
  console.log("Iteration: " + i);
}

This code will print the message 5 times.
