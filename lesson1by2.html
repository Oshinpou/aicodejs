Lesson 1.2: Variables & Data Types

In JavaScript, variables are containers for storing data values. You can use let, const, or the older var keyword to declare them.

JavaScript Data Types

  • String — Text (e.g., "Hello")
  • Number — Integers and decimals (e.g., 42, 3.14)
  • Boolean — true or false
  • Undefined — a declared variable without a value
  • Null — an empty or non-existent value
  • Object — collections of key/value pairs
  • Array — ordered lists of values

Example Code

// Declaring variables
let name = "Alice";
const age = 30;
var isStudent = true;

// Output to console
console.log("Name:", name);
console.log("Age:", age);
console.log("Student?", isStudent);
