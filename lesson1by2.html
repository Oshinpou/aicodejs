Lesson 2: Variables & Data Types

Understand how AI handles JavaScript variables and data. Learn how to manage different types of data effectively.

Code Editor

Examples and Explanations

Example 1: Declaring a Variable 
let name = "AIcodeJS";

Explanation: We are creating a variable named name and assigning a string value to it.

Example 2: Working with Numbers 
let number = 100 + 50;

Explanation: Performing addition and storing the result.

Example 3: Boolean Value 
let isAIReady = true;

Explanation: Storing a true/false value in a variable.

Example 4: Using Template Literals 
let greeting = `Hello, ${name}`;

Explanation: Embedding variables inside a string using backticks.

Example 5: Type Checking 
console.log(typeof name);

Explanation: Checking the data type of a variable.

Example 6: Array of Items 
let fruits = ["apple", "banana", "cherry"];

Explanation: Storing multiple values in one variable.

Example 7: Object Example 
let car = { brand: "Tesla", model: "S", year: 2022 };

Explanation: Grouping related data together in an object.

Example 8: Null and Undefined 
let empty = null; let notAssigned;

Explanation: `null` is intentionally empty; `undefined` means no value assigned yet.

Example 9: Constant Variables 
const PI = 3.14159;

Explanation: Constant variables cannot be reassigned once declared.

Example 10: Boolean Logic 
let isAdult = age >= 18;

Explanation: Using comparison operators to create boolean values.
