Example 1: Creating and Using Objects

const person = {
    name: 'John',
    age: 30,
    greet: function() {
        return 'Hello, ' + this.name;
    }
};

console.log(person.greet()); // Output: "Hello, John"
Explanation: This example shows how to create an object with properties and methods in JavaScript. We define a `person` object with `name`, `age`, and a method `greet` that returns a greeting string using the `name` property.

Example 2: Working with Arrays

const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(function(num) {
    return num * 2;
});

console.log(doubled); // Output: [2, 4, 6, 8, 10]
Explanation: This example demonstrates how to use the `map()` method to double each number in the array. The `map()` method returns a new array with the results of calling the provided function on every element.

Example 3: Object Destructuring

const person = { name: 'John', age: 30 };
const { name, age } = person;

console.log(name); // Output: "John"
console.log(age);  // Output: 30
Explanation: This example illustrates how to use object destructuring to extract the properties `name` and `age` from the `person` object into their own variables.

Example 4: Array Destructuring

const numbers = [10, 20, 30];
const [first, second] = numbers;

console.log(first);  // Output: 10
console.log(second); // Output: 20
Explanation: This example demonstrates array destructuring. We assign values from the `numbers` array into the `first` and `second` variables.
