Lesson 9: DOM Manipulation

1. Selecting Elements by ID

Use document.getElementById() to select elements by their ID.

document.getElementById("element-id");

Example:

This is a paragraph.

2. Selecting Elements by Class Name

Use document.getElementsByClassName() to select elements by their class name.

document.getElementsByClassName("class-name");

Example:

This is another paragraph with a class.

3. Modifying Element Styles

Change the styles of elements using JavaScript by modifying the style property.

document.getElementById("element-id").style.color = "red";

Example:

This text will change color.

4. Changing HTML Content

Use innerHTML to change the content inside an HTML element.

document.getElementById("element-id").innerHTML = "New Content";

Example:

This content will be updated with your input.

5. Adding Multiple Event Listeners

Learn to add multiple event listeners to an element.

element.addEventListener("event1", function1);
element.addEventListener("event2", function2);
