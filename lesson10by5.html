Lesson 10.5: Input Events for Smart Forms

Input events help build smart forms where AI suggestions, feedback, or validations are provided in real-time as users type.

Example: AI Smart Greeting

AI is waiting for your input...

JavaScript Code


const nameInput = document.getElementById("nameInput");
const responseText = document.getElementById("responseText");

nameInput.addEventListener("input", function(e) {
  const name = e.target.value.trim();
  if (name.length > 0) {
    responseText.textContent = `Hello, ${name}! Let's get started with AI.`;
  } else {
    responseText.textContent = "AI is waiting for your input...";
  }
});
