Lesson 8.6: Object Destructuring

1. What is Object Destructuring?

Object destructuring is a concise way to extract values from objects and assign them to variables. It's very useful in AI data structures, where data is often deeply nested.

2. Example: Destructuring AI Configuration

const aiConfig = {
  model: "VisionAI",
  settings: {
    resolution: "4K",
    fps: 30
  }
};

// Destructuring
const { model, settings: { resolution } } = aiConfig;
console.log(model);       // VisionAI
console.log(resolution);  // 4K
