Lesson 5.6: Nested Loops for Complex AI Tasks

What Are Nested Loops?

Nested loops are loops inside loops. In AI, theyâ€™re used to process multi-dimensional data such as tensors, confusion matrices, or to simulate multiple model parameters.

Basic Nested Loop Example

for (let epoch = 1; epoch <= 3; epoch++) {
  console.log("Epoch:", epoch);
  for (let batch = 1; batch <= 2; batch++) {
    console.log("  Processing batch", batch);
  }
}

Use Case: Confusion Matrix

const confusionMatrix = [
  [25, 3, 2],
  [4, 30, 1],
  [0, 2, 35]
];

for (let i = 0; i < confusionMatrix.length; i++) {
  for (let j = 0; j < confusionMatrix[i].length; j++) {
    console.log(`Class ${i} vs ${j}: ${confusionMatrix[i][j]}`);
  }
}

Best Practices

  • Be cautious of performance in deeply nested loops.
  • Break logic into reusable functions if loops become too complex.
  • Use meaningful variable names like epoch, batch, i, j.
