Lesson 15.1 — Introduction to Scope

Understand how AI accesses and restricts variables in logic using scope.

What is Scope?

In JavaScript (and AI logic), scope defines where variables can be accessed. There are three main types:

  • Global Scope: Accessible everywhere in the code.
  • Function Scope: Accessible only inside a function.
  • Block Scope: Created with let and const, restricted to blocks like { }.

Examples of Scope

// Global scope
let aiName = "CosmoAI";

function runAI() {
  // Function scope
  let task = "learning";
  console.log(aiName); // Can access global
  console.log(task);   // Can access local
}

runAI();
// console.log(task); // Error: task is not defined (outside function)

{
  // Block scope
  const status = "active";
  console.log(status);
}
// console.log(status); // Error: status is not defined (outside block)

Why Scope Matters in AI

In AI applications, scope helps isolate models, parameters, and logic without interference. Proper use ensures secure, modular AI systems.
