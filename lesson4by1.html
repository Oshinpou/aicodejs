Example 1: Basic AI Logic Using if-else

let userInput = "happy";
if (userInput === "happy") {
  console.log("AI says: Glad you're happy!");
} else {
  console.log("AI says: Let's improve your mood.");
}

Description: This AI checks the user's emotion and gives a response.
