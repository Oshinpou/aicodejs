Lesson 4.1: Advanced Functions

Learn closures, currying, and higher-order functions in JavaScript to enhance AI logic and efficiency.

What You'll Learn:

This lesson covers advanced concepts of JavaScript functions that are crucial for writing efficient, reusable, and modular code in AI applications.

Closures

A closure is a function that retains access to its lexical scope, even when the function is executed outside that scope. Let's look at an example:


function outer() {
  let counter = 0;

  return function inner() {
    counter++;
    console.log(counter);
  }
}

const increment = outer(); // 'increment' is the closure
increment(); // Output: 1
increment(); // Output: 2

In the above example, the inner function retains access to the variable 'counter' defined in the outer function, even after the outer function has executed.

Function Currying

Currying is the technique of transforming a function that takes multiple arguments into a sequence of functions, each taking a single argument. Here's an example:


function multiply(a) {
  return function(b) {
    return a * b;
  }
}

const multiplyBy2 = multiply(2);
console.log(multiplyBy2(5)); // Output: 10

In the above example, the multiply function is curried, and we create a new function 'multiplyBy2' that always multiplies any input by 2.

Higher-Order Functions

A higher-order function is a function that either takes one or more functions as arguments, returns a function as a result, or both. For example:


function processData(data, operation) {
  return operation(data);
}

function square(x) {
  return x * x;
}

console.log(processData(5, square)); // Output: 25

Here, 'processData' is a higher-order function that accepts a function ('square') as an argument and applies it to the data.
