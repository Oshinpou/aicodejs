Lesson 2.5: DOM Events

Handle events and build dynamic user interactions

What are DOM Events?

Events are actions or occurrences that happen in the browser, such as clicks, mouse movements, key presses, and more. JavaScript can listen for and respond to these events using event listeners.

Basic Event Listening

document.getElementById("btn").addEventListener("click", function() {

alert("Button clicked!"); });

Common Event Types

  • click
  • mouseover
  • keydown
  • submit

Task

Create a form with an input field and a submit button. When the form is submitted, prevent the page from reloading and show the entered text in an alert.
