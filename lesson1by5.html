Lesson 1.5: Loops

while, for, and loop control in AI code

Loops allow you to repeat a block of code multiple times. In AI programming, you might loop through data sets, repeat training processes, or iterate over input combinations.

1. while Loop

The while loop runs a block of code as long as a condition is true.

let count = 0; while (count < 3) { console.log("AI Cycle " + count); count++; }

2. for Loop

The for loop gives you more control with an initializer, condition, and incrementer.

for (let i = 0; i < 3; i++) { console.log("Training round: " + i); }

3. Loop Control: break and continue

Use break to exit a loop early. Use continue to skip the current iteration.

for (let i = 0; i < 5; i++) { if (i === 2) continue; if (i === 4) break; console.log(i); }
