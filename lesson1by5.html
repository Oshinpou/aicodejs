Task 1: Basic Loop

Write a simple `for` loop that counts from 1 to 5 and prints each number to the console.

for (let i = 1; i <= 5; i++) {
    console.log(i);
}

Quiz 1: Loop Basics

What is the purpose of a loop in programming?

1. To repeat tasks until a condition is met.

2. To run only once.

3. To create an infinite program.

Answer: 1. To repeat tasks until a condition is met.

Example 1: While Loop

In this example, a `while` loop is used to count down from 10 to 1.

let count = 10;
while (count >= 1) {
    console.log(count);
    count--;
}

Task 2: While Loop

        
Write a `while` loop that sums numbers from 1 to a user input value and prints the total.

        

            
let sum = 0;
let number = prompt("Enter a number to sum up to:");
let i = 1;
while (i <= number) {
    sum += i;
    i++;
}
console.log("The total sum is: " + sum);
            

        

    


    

        
Example 2: Do...While Loop

        
The `do...while` loop is guaranteed to execute at least once, even if the condition is false at the start.

        

            
let number = 0;
do {
    number = prompt("Enter a number greater than 10:");
} while (number <= 10);
console.log("You entered a valid number: " + number);

Task 5: Break and Continue

        
Write a loop that prints numbers from 1 to 10, but skips numbers that are divisible by 3.

    if (i % 3 === 0) {
        continue;
    }
    console.log(i);
}

Task 6: Array Iteration

        
Write a function that takes an array of names and returns the list of names that have more than 5 characters using a loop.

        

            
let names = ["Alice", "Bob", "Charlie", "David"];
let longNames = [];

for (let i = 0; i < names.length; i++) {
    if (names[i].length > 5) {
        longNames.push(names[i]);
    }
}
console.log(longNames);

Task 7: Factorial Calculation

        
Write a loop to calculate the factorial of a number entered by the user.

        

            
let number = prompt("Enter a number:");
let factorial = 1;

for (let i = 1; i <= number; i++) {
    factorial *= i;
}

console.log("The factorial of " + number + " is: " + factorial);
