Lesson 8.3: Constructor Functions for Objects

1. What is a Constructor Function?

A constructor function is used to create multiple instances of objects with the same structure. This is useful in AI systems for generating multiple models, agents, or datasets with different attributes.

2. Creating a Constructor Function

function AIAgent(name, skill) {
  this.name = name;
  this.skill = skill;
  this.activate = function() {
    console.log(this.name + " activated with skill: " + this.skill);
  };
}

let agent1 = new AIAgent("Alpha", "Image Recognition");
agent1.activate();

3. Real Use in AI Apps

function AIModel(modelName, trained) {
  this.modelName = modelName;
  this.trained = trained;
  this.train = function() {
    this.trained = true;
    console.log(this.modelName + " is now trained.");
  };
}

let visionAI = new AIModel("VisionNet", false);
visionAI.train();
