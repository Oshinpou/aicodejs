Lesson 6.2: Function Parameters & Arguments

What Are Parameters & Arguments?

Parameters are placeholders in a function definition. Arguments are the actual values passed to the function when itâ€™s called. In AI, they allow us to feed different data into reusable logic blocks.

Example: Activation Score Function

function activationScore(input, weight) {
  return input * weight;
}

let score = activationScore(0.75, 1.2);
console.log("Score:", score);

This allows us to reuse the same function for any `input` and `weight` values, just like in a neural net.
