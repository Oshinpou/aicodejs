Lesson 5.3: While Loops for Dynamic Data Processing

What is a while loop?

A while loop repeats a block of code as long as a condition is true. It is great for tasks where the number of repetitions is not fixed in advanceâ€”useful in AI processing where data changes over time.

Basic Syntax

let i = 0;
while (i < 5) {
  console.log("AI iteration:", i);
  i++;
}

AI Example: Processing User Data Until Found

const users = ["Anna", "Ben", "Clara", "David"];
let index = 0;
while (users[index] !== "Clara") {
  console.log("Searching... Found:", users[index]);
  index++;
}
console.log("User Clara found at index", index);

Key Points

  • Use while when the end condition is not fixed.
  • Make sure the condition will eventually become false, or you'll create an infinite loop.
  • Common in AI tasks such as data search, simulation, and iterative computation.
