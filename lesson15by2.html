Lesson 15.2 — Global Scope in AI

How global variables impact AI logic and behavior in JavaScript.

What is Global Scope?

A variable is in the global scope if it is declared outside any function or block. It can be accessed or modified from anywhere in the program, including inside functions or AI routines.

Example: Global Scope

// Global variable
let aiMode = "learning";

function processAI() {
  console.log("AI mode is:", aiMode); // Accessing global variable
}

function updateMode() {
  aiMode = "predicting"; // Modifying global variable
}

processAI();
updateMode();
processAI();

AI Insight

Global variables are powerful but risky in AI systems. Overusing them can lead to unexpected behavior, especially when multiple components access or modify shared values. For scalable AI logic, keep most variables local and use global scope only for constants or configuration settings.
