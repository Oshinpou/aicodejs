Lesson 5.2: Using for Loops to Iterate Over Data

What is a for loop?

A for loop allows you to repeat a block of code a specific number of times. It's especially useful when working with arrays or lists.

for (let i = 0; i < 5; i++) {
  console.log("Hello AI, loop number " + i);
}

Use Case: Looping Through an Array

Let's print every item in an array using a for loop.

const colors = ["red", "blue", "green"];
for (let i = 0; i < colors.length; i++) {
  console.log(colors[i]);
}

Explanation

  • for (let i = 0; i < 5; i++) has 3 parts: initialize, condition, increment.

  • i++ increases the loop variable by 1 each time.
