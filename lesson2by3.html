Lesson: Operators & Expressions

Explore JavaScript's operators and learn how to build smart expressions!

100 Features Covered
  • Basic Arithmetic: +, -, *, /, %, **
  • Assignment Operators: =, +=, -=, *=, /=, %=, **=
  • Comparison Operators: ==, ===, !=, !==, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Bitwise Operators: &, |, ^, ~, <<, >>, >>>
  • String Operators: +, +=
  • Type Operators: typeof, instanceof
  • Unary Operators: +, -, !, ++, --, typeof, delete, void
  • Ternary Operator: ? :
  • Comma Operator: ,
  • Destructuring: Arrays and Objects
  • Spread Operator: [...arr], {...obj}
  • Rest Operator: ...rest
  • Optional Chaining: ?. ?.()
  • Nullish Coalescing: ??
  • In Operator: "key" in object
  • Delete Operator
  • New Operator
  • This Operator
  • Instanceof Operator

  • Grouped Expressions with ()

  • Operator Precedence
  • Operator Associativity
Coding Examples

1. Arithmetic Operators

let sum = 5 + 3; // 8 let product = 4 * 2; // 8

2. Assignment Operators

let x = 10; x += 5; // x is now 15

3. Comparison Operators

5 == "5"; // true 5 === "5"; // false

4. Logical Operators

true && false; // false true || false; // true !true; // false

5. String Operators

let greeting = "Hello" + " World!";

6. Ternary Operator

let age = 18; let canVote = (age >= 18) ? "Yes" : "No";

7. Spread and Rest

let numbers = [1, 2, 3]; let moreNumbers = [...numbers, 4, 5];

8. Nullish Coalescing Operator

let username = null; let displayName = username ?? "Guest"; // Guest

9. Optional Chaining

let user = { profile: { name: "Alex" } }; console.log(user?.profile?.name); // Alex
