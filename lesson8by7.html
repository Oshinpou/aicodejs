Lesson 8.7: Object Spread and Rest

1. What is Spread?

The spread operator (...) allows you to expand the properties of an object. It's useful when copying or combining objects in AI configuration or dataset management.

const baseModel = { name: "AI-X", version: "1.0" };
const extendedModel = { ...baseModel, accuracy: "98%" };

console.log(extendedModel);
// { name: 'AI-X', version: '1.0', accuracy: '98%' }

2. What is Rest?

The rest operator (...) is used to collect the remaining properties into a new object. This is useful when separating important AI model parameters from secondary ones.

const modelData = { name: "NeuroNet", speed: "fast", type: "CNN" };
const { name, ...details } = modelData;

console.log(name);    // NeuroNet
console.log(details); // { speed: 'fast', type: 'CNN' }
