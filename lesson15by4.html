Lesson 15.4 — Function Scope vs Block Scope

Understand where your AI variables live and why it matters.

Function Scope

Variables declared with var are function-scoped, meaning they are only accessible within the function they are declared in.

function aiProcess() {
  var model = "NeuralNet";
  console.log(model); // Works
}
console.log(model); // Error: model is not defined

Block Scope

Variables declared with let and const are block-scoped, meaning they only exist inside the curly braces { } where they are defined.

{
  let layer = "Convolution";
  console.log(layer); // Works
}
console.log(layer); // Error: layer is not defined

AI Insight

For secure and modular AI design, use block-scoped variables (let and const) to prevent accidental data access across unrelated processes.
