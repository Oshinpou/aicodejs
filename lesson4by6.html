Lesson 4.6: Nested Conditional Statements

Overview

Nested conditional statements allow an AI program to make decisions within decisions. This means one condition may only be checked if another is true, providing more advanced control flow.

Example Explanation

Below is a nested if-else structure to evaluate age and membership level of a user to determine eligibility for AI access levels.

const age = 25; const membership = "premium";if (age >= 18) { if (membership === "premium") { console.log("Access granted to premium AI features."); } else { console.log("Access granted to basic AI features."); } } else { console.log("Access denied. You must be 18 or older."); }
