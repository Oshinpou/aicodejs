Lesson 7.7: Using Arrays for AI Data Models

1. Introduction

Arrays in JavaScript are powerful tools to manage AI data structures such as datasets, label arrays, model parameters, and predictions.

2. Example: AI Dataset

let inputData = [0.2, 0.5, 0.7, 0.9];
let labels = [0, 1, 1, 0];

Here, inputData can represent AI input features and labels the target output for training.

3. Structuring Input and Output

let dataset = [
  { input: [0.1, 0.2], output: 0 },
  { input: [0.4, 0.6], output: 1 }
];

This structure helps simulate labeled data points for training neural networks or classifiers.

4. Iterating Over Dataset

dataset.forEach((data, index) => {
  console.log("Input:", data.input, "Output:", data.output);
});

5. AI Simulation: Predicting with Dummy Logic

function predict(input) {
  let sum = input.reduce((acc, val) => acc + val, 0);
  return sum > 0.5 ? 1 : 0;
}

Using an array to simulate input passing through a simple prediction logic.
