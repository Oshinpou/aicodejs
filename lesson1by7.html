Lesson 1.7 – Objects & Arrays

Objects and arrays are essential structures in JavaScript to store and manipulate data efficiently, especially in AI applications.

What is an Array?

An array holds a list of values.

const fruits = ["apple", "banana", "mango"];
console.log(fruits[1]); // Output: banana

What is an Object?

An object stores data as key-value pairs.

const student = {
  name: "John",
  age: 22,
  course: "AI & JS"
};

console.log(student.name); // Output: John

Combining Arrays & Objects

const users = [
  { name: "Alice", score: 85 },
  { name: "Bob", score: 90 }
];

console.log(users[1].score); // Output: 90
