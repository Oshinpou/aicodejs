Understanding Arrays & Iteration

In JavaScript, arrays allow you to store multiple values in a single variable. Iteration is the process of looping through the array and accessing its elements. Understanding these concepts is crucial for creating efficient AI programs.

Coding Example 1: Iterating Through an Array

Code:

const numbers = [1, 2, 3, 4, 5];
for (let i = 0; i < numbers.length; i++) {
    console.log(numbers[i]);
}

Explanation:

This code declares an array of numbers and uses a for loop to iterate through each element in the array. It then prints each number to the console.

Coding Example 2: Iterating Through an Array with forEach

Code:

const fruits = ["Apple", "Banana", "Cherry", "Date"];
fruits.forEach(fruit => {
    console.log(fruit);
});

Explanation:

The forEach method is another way to iterate through an array. It takes a function that gets called for each element in the array.

Coding Example 3: Using map() for Array Transformation

Code:

const numbers = [1, 2, 3, 4];
const squaredNumbers = numbers.map(num => num * num);
console.log(squaredNumbers);

Explanation:

The map() method creates a new array by applying a function to each element. In this example, we square each number in the array.
