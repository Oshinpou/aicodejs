Lesson 1.4: Control Flow

Mastering Conditions and Logic

Control flow in JavaScript determines how code is executed based on different conditions. It helps your AI logic make decisions and react differently under varying inputs.

if, else if, and else

let score = 85;if (score >= 90) { console.log("Excellent"); } else if (score >= 75) { console.log("Very Good"); } else { console.log("Keep Practicing"); }

switch Statement

let grade = 'B';switch (grade) { case 'A': console.log("Outstanding"); break; case 'B': console.log("Great Job"); break; default: console.log("Try Again"); }

Task:

Create a simple program that asks a user for temperature and logs a message:

  • If temp > 30: "It's hot outside."
  • If temp between 15-30: "It's pleasant."
  • Else: "It's cold."

Quiz:

  1. What does the break statement do in a switch block?
  2. How is else if different from if?
  3. What will be the output of:
    let x = 5; if (x > 3) console.log("Hi");

Summary:

Control flow structures such as if-else and switch help guide how your JavaScript code makes decisions, which is critical in building intelligent behaviors in AI code.
